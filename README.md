每隔一定时间清理配置文件里的NPC和物品。

[插件直接下载地址](https://gitee.com/hufang360/TShockDisableNPC/attach_files/1049368/download/DisableNPC.dll)

配置文件位于 ./tshock/DisableNPC.json，如果对应位置没有配置文件，插件会自动创建，并默认添加大部分NPC和合成站进白名单。
配置示例：
```json
{
  "npc": [19,106],
  "item": [],
  "second": 2
}
```
